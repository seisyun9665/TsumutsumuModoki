# TsumuTsumuModoki

## 概要
このプロジェクトは、Unity製のパズルゲーム「TsumuTsumuModoki」のソースコードです。同じ種類のフルーツを繋げて消していく、シンプルで親しみやすいカジュアルゲームです。

## デモプレイ
以下のリンクからゲームをプレイすることができます：
https://unityroom.com/games/tsumutsumumodoki

## 技術的なポイント
- **オブジェクト指向設計**: フルーツやボムなどの各要素をコンポーネントとして分離し、保守性の高いコード設計を実現
- **シングルトンパターン**: ゲーム状態管理にシングルトンパターンを採用し、効率的なデータ管理を実現
- **最適化**: モバイル向けのパフォーマンス最適化を実施
- **UI/UX**: 直感的な操作感とフィードバックを重視したUI設計

## 動機と目標達成
本プロジェクトは、[G2Studioインターンシップ](https://note.g2-studios.net/n/n71d814f6b5ff#ebfde5ac-c3e9-481a-96a9-932c57a4b4fc)

- **成果**: 11日間という短期間で、基本機能の実装からデプロイまでを完遂

### 学んだこと
- 短期開発における優先順位付けの重要性
- Unity基礎技術（コンポーネント設計、物理演算、UI実装など）
